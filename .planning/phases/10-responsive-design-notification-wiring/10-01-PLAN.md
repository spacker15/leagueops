**Part A: AppShell RightPanel bottom drawer (MOB-02 per D-04 through D-07)**

IMPORTANT: Use conditional rendering, NOT CSS `hidden lg:block`, to prevent double Supabase subscriptions. RightPanel subscribes to realtime channels — rendering it hidden still mounts the component and opens subscriptions.

1. **Add imports** at top of AppShell.tsx:

   ```typescript
   import { Cloud, X as XIcon } from 'lucide-react'
   ```

2. **Add drawer state and media query hook** immediately after the existing `const [activeTab, setActiveTab] = useState<TabName>(...)` on line 60 (BEFORE the loading guard at line 103):

   ```typescript
   const [drawerOpen, setDrawerOpen] = useState(false)
   const [isLg, setIsLg] = useState(true) // default to true for SSR/initial render

   useEffect(() => {
     const mql = window.matchMedia('(min-width: 1024px)')
     setIsLg(mql.matches)
     const handler = (e: MediaQueryListEvent) => {
       setIsLg(e.matches)
       if (e.matches) setDrawerOpen(false) // close drawer when resizing to desktop
     }
     mql.addEventListener('change', handler)
     return () => mql.removeEventListener('change', handler)
   }, [])
   ```

3. **Replace the existing RightPanel** (line 163: `<RightPanel onNavigate={setActiveTab} />`) with a conditional render. Replace line 163 with:

   ```tsx
   {
     isLg && <RightPanel onNavigate={setActiveTab} />
   }
   ```

   This ensures RightPanel only mounts at lg+ (1024px+). Below lg, it is not rendered at all — no Supabase subscriptions, no wasted resources.
